# HackMe April Track Topics

## 1. Cipher Hunters

- Format: 3-level cryptography track with increasing difficulty.
- Level 1: classical cipher structure on natural-language literature text.
- Level 2: classical cipher with extra twist that makes direct pattern matching harder.
- Level 3: AI-flavored layered cipher, likely with tokenization-inspired step and length-obscuring encoding.
- Source text: literature excerpt / poem / chapter text; exact text is not fixed in the site content.
- Goal: decipher the text and document hypotheses, scripts, rejected ideas, and final reasoning.

## 2. AutoGrad Scientist

- Framework: `TUES-AI/AutoGrad`.
- Dataset ladder:
  - Level 1: Banknote Authentication
  - Level 2: Seeds
  - Level 3: Digits
- Goal: build baselines, run controlled experiments, compare changes, and document findings.

## 3. Learning Enemy

- Core idea: make a small game where the enemy actually learns from player behavior.
- Allowed main constraint modes:
  - `13 KB zipped browser game`
  - `300 LOC no-engine`
- Learning must be visible: replay, log, side panel, GIF, or similar trace.
- Goal: show real adaptation, not fake difficulty scaling.

## 4. Scratch for ML

- Core idea: build an interactive ML playground inspired by `TensorFlow Playground`.
- Example directions:
  - mini neural network playground on a 2D dataset
  - Scratch-like block builder for a tiny model
  - educational visualization for loss / learning rate / features / noise
- Goal: make ML behavior understandable through interaction, sliders, visuals, and live changes.
